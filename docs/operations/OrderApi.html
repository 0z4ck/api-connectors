order_newOrder

URL

https://www.bitmex.com/api/v1/order/new

HTTP Method

POST

Response Type

order

Parameters

    symbol : string (form)

    Instrument name.

quantity : double (form)

Quantity. Use positive numbers to buy, negative to sell.

price : double (form)

Price to buy.

ioc : boolean (form)
optional

Set to true to place an immediateOrCancel order.
